*Graphics Script For Idiots*
Pupose: Create a graphical scripting language that an idiot can use. Compiles to Javascript for use with WebGL

Grammar: 
Command <arguments>

Features:
 * Easy to read
 * Quick to learn
 * No conditionals (yet)
 * WebGL Integration
 * Not turing complete 
Behind the Scenes:
 * Poorly organized code
 * Two phase interpretations
 * Pseudo-tokenization
